# ภาพปุ่มบริการหน้าแรก (3 การ์ด)

สไตล์ **medium-tone proptech card** — รูปเต็มการ์ด (ซ้ายว่าง / ขวาภาพประกอบ) + ข้อความขาวจาก Flutter ทับพื้นที่ว่างซ้าย

| ไฟล์ | ปุ่ม | Gradient |
|------|------|----------|
| `home_service_map.png` | ค้นหาแผนที่ | `#6B4EAA` → `#8B6FD4` |
| `home_service_match.png` | ช่วยหาทรัพย์ฟรี | `#0F9B8E` → `#2DD4BF` |
| `home_service_board.png` | บอร์ดหาทรัพย์ | `#E85A00` → `#FF9A4D` |

## ดีไซน์

- รูป: เต็มการ์ด — ซ้าย ~58% พื้นที่ว่างสำหรับข้อความ, ขวาภาพประกอบ
- ข้อความ: **Flutter widget** ทับบนรูป — `fontSize: 12`, `fontWeight: w800`, สีขาว (เท่ากันทุกการ์ด)
- **ไม่ทาสีทับรูป** และ **ไม่ฝัง** ข้อความใน asset

## ขนาด

| รายการ | ค่า |
|--------|-----|
| ความสูงการ์ด | **60 pt** |
| ข้อความ | **11.5 pt** w600 — 2 บรรทัด |
| พื้นที่ข้อความ | **58%** ความกว้างการ์ด (ซ้าย) |
| อัตราส่วนรูป asset | **3:2** (1536×1024) |
