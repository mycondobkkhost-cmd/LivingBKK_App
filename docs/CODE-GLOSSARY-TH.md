## หลักกันสับสน

1. **RXT** = ทะเบียนกลาง (หลายโพสต์ → หนึ่งทรัพย์)
2. **RENT/SALE** = ประกาศแต่ละโพสต์
3. **มอบสิทธิ์ดูแล** ≠ เปลี่ยนเจ้าของกฎหมาย — คือให้คนนั้นดูแลในแอป
4. ใน UI ทุกรหัสควรมี **คำอธิบายไทย** จาก `CodeGlossary.captionFor()`
