# คู่มือ — ใช้ LivingBKK บนมือถือ (ไม่ต้องเขียนโค้ด)

## แอปนี้คืออะไร

**LivingBKK** คือเว็บแอปหาบ้าน/คอนโดในกรุงเทพ — เปิดผ่านเบราว์เซอร์บนมือถือได้ ไม่จำเป็นต้องลงจาก App Store ก่อน

---

## วิธีที่ 1: เปิดบนมือถือในบ้าน (ทดลอง) — แนะนำ

### กรณี Mac ใช้เน็ตจาก **ฮอตสปอตมือถือ** (แชร์จาก iPhone/Android)

มือถือกับ Mac **อยู่คนละบทบาท** แต่ยังลองได้ — ใช้สคริปต์นี้บน Mac:

```text
./scripts/run-for-phone.sh
```

จอ Terminal จะบอกลิงก์บนมือถือ เช่น `http://172.20.10.2:8084`  
→ **คัดลอกลิงก์นั้นไปเปิดใน Safari หรือ Chrome บนมือถือ** (อย่าใช้ localhost)

ถ้าเปิดไม่ได้: Mac → System Settings → **Network** → ดูชื่อ iPhone/USB → **Details** → จำเลข **IP Address** แล้วเปิด `http://เลข-IP:8084`

### กรณี Mac กับมือถือใช้ **Wi‑Fi บ้านตัวเดียวกัน**

1. บน Mac รัน `./scripts/run-for-phone.sh`
2. บนมือถือเปิดลิงก์ที่ Terminal แสดง (เช่น `http://192.168.1.25:8084`)

### กรณีรันแบบเดิม (ดูบน Mac อย่างเดียว)

```text
./scripts/run-app.sh
```

ลิงก์ `localhost` **ใช้บนมือถือไม่ได้** — ใช้เฉพาะบน Mac

---

## วิธีที่ 2: ติดไอคอนบนจอโฮม (เหมือนมีแอป)

### iPhone (Safari)

1. เปิดลิงก์แอปใน Safari  
2. กดปุ่ม **แชร์** (สี่เหลี่ยมมีลูกศรขึ้น)  
3. เลือก **เพิ่มไปที่หน้าจอโฮม**  
4. กด **เพิ่ม** — จะมีไอคอน LivingBKK บนจอโฮม

### Android (Chrome)

1. เปิดลิงก์ใน Chrome  
2. เมนู ⋮ → **เพิ่มไปยังหน้าจอหลัก** หรือ **Install app**

---

## วิธีที่ 3: ให้คนอื่นเปิดทางอินเทอร์เน็ต

ต้องมีคนช่วย **สร้างเว็บ** แล้วอัปโหลด:

1. รัน `./scripts/build-web.sh` บน Mac  
2. ได้โฟลเดอร์ `mobile/build/web`  
3. ลากโฟลเดอร์นั้นไปวางที่ [Netlify Drop](https://app.netlify.com/drop) (ฟรี ไม่ต้องสมัครก็ได้ชั่วคราว)  
4. ได้ลิงก์ `https://....netlify.app` ส่งให้คนอื่นเปิดบนมือถือ

ก่อนอัปโหลด ต้องตั้ง **Supabase** ให้แอปเชื่อมข้อมูลจริง — ดู [QUICKSTART-ONE-SHOT.md](QUICKSTART-ONE-SHOT.md)

## คำถามบ่อย

**ทำไมแผนที่ไม่ขึ้น?**  
อาจยังไม่ได้ใส่คีย์ Google Maps — แอปจะใช้แผนที่ฟรี (OpenStreetMap) แทน ยังใช้งานได้

**เลื่อนลงไม่ได้?**  
เลื่อนนิ้วขึ้น-ลงบนหน้าจอ — รายการทรัพย์อยู่ด้านล่างแผนที่

**โหมด Demo คืออะไร?**  
ยังไม่ต่อคลังข้อมูลจริง แอปใช้บ้านตัวอย่างในเครื่องให้ลองเล่น
