# คู่มือรวบรวมชื่อโครงการ (ทาง A) — ใช้งานง่าย ไม่ต้องเขียนโค้ด

**เป้าหมาย:** ให้ตอนลงประกาศ พิมพ์ชื่อโครงการแล้ว**เจอ** (เหมือนแอปอื่น) โดย**ไม่จ่าย Google**

**หลักการ:** เก็บชื่อโครงการลง **สมุดกลางบนอินเทอร์เน็ต (Supabase)** → แอปโหลดสมุดนั้นมาใช้

---

## ก่อนเริ่ม — ทำครั้งเดียว (ให้คนที่ช่วยคุณ 15–30 นาที)

1. แอปต่อ **Supabase Cloud** แล้ว (`mobile/assets/env` มี URL + key)
2. รันคำสั่ง (บน Mac โปรเจกต):

```bash
source scripts/dev-path.sh
./scripts/deploy-projects-cloud.sh
```

3. ล็อกอินแอปเป็น **แอดมิน** → ไปแท็บ **โครงการ** (ไม่ใช่แท็บ「นำเข้า」)

ถ้าขั้นนี้ยังไม่เคยทำ — ดึงโครงการจากเว็บในแอปจะ**ไม่บันทึกจริง**

---

## อย่าสับสน 2 แท็บ

| แท็บในแอดมิน | ใส่อะไร | ใช้ตอนลงประกาศไหม |
|-------------|---------|-------------------|
| **นำเข้า** | ลิงก์ **ประกาศ** ทรัพย์ (LI) | ไม่ใช่ชื่อโครงการ |
| **โครงการ** | **ชื่อโครงการ** | **ใช่ — นี่แหละ** |

---

## วิธีเพิ่มโครงการ (เลือก 1 หรือทำคู่กัน)

### วิธีที่ 1 — ง่ายสุด: ทีละลิงก์ Property Hub (เหมาะโครงการใหม่)

1. เปิด **Property Hub** → หน้าโครงการ
2. คัดลอก **ลิงก์หน้าโครงการ** (ไม่ใช่ลิงก์ประกาศห้องเดียว)  
   รูปแบบ: `https://propertyhub.in.th/projects/...`
3. แอดมิน → **โครงการ** → วางลิงก์ → กด **ดึงจากลิงก์**
4. รอข้อความสำเร็จ → ชื่อโครงการจะโผล่ในรายการด้านล่าง

ทำซ้ำทุกครั้งที่เจอโครงการใหม่

---

### วิธีที่ 2 — เร็วสุด: กดปุ่มเดียว (Property Hub เว็บเดียว)

1. แอดมิน → **โครงการ**
2. กด **ดึงทั้งหมดจาก Property Hub** (รอจนจบ — อาจ 10–30 นาที)
3. ดูข้อความสรุปว่าสำเร็จกี่รายการ / ล้มเหลวกี่รายการ

รายละเอียดเพิ่ม: `docs/คู่มือ-Property-Hub-เว็บเดียว.md`

---

### วิธีที่ 3 — มือ: กดปุ่ม + ที่มุมขวาล่าง

1. กด **เพิ่มโครงการ**
2. กรอกอย่างน้อย: **ชื่อไทย**, **เขต**, **ละติจูด/ลองจิจูด** (หรือเลือกจากแผนที่ถ้ามี)
3. บันทึก

ใช้เมื่อเว็บอื่นไม่มีลิงก์ให้ดึงอัตโนมัติ

---

## รวบรวมแบบไม่ตกหล่น — ใช้ “แผนที่เขต”

อย่าดึงแบบสุ่ม ให้ไล่เป็น **โซน** แล้วติ๊กเช็กลิสต์

### รอบที่ 3 — โครงการที่คนมักพิมพ์ผิด / ชื่อเล่น

หลังดึงแล้ว ให้คนทีมเพิ่ม **ชื่อเล่น** ในแอดมิน (แก้โครงการ → ช่อง alias) เช่น

- ทรู ทองหล่อ ↔ The Trendy
- ไอดีโอ โมบิ ↔ Ideo Moby ฯลฯ

---

## เช็กว่า “เข้าแอปแล้ว” (ทุกสัปดาห์)

### ในแอดมิน

- แท็บ **โครงการ** → ดูตัวเลข **X / Y โครงการ**
- ค้นหาชื่อที่เพิ่งดึง เช่น `ทรู` ต้องเจอในรายการ

### ใน Supabase (ถ้ามีคนช่วย)

```sql
SELECT count(*) FROM property_projects WHERE is_active = true;
```

เลขควร**เพิ่มขึ้น**ทุกครั้งหลังดึงชุดใหญ่

### ทดสอบเหมือนผู้ใช้จริง

1. ปิดแอปเปิดใหม่ (หรือรอ 1 นาที)
2. ไป **ลงประกาศ** → ช่อง **ค้นหาโครงการ**
3. พิมพ์ชื่อที่เพิ่งเพิ่ม → ต้องขึ้นในรายการ

## ตารางงานรายสัปดาห์ (ไม่ตกหล่น)

| วัน | งาน | เวลาโดยประมาณ |
|-----|-----|----------------|
| จันทร์ | ไล่โซน BTS 1 โซน (ดึงลิงก์หรือชุดใหญ่) | 30–60 นาที |
| พุธ | ไล่โซน BTS อีก 1 โซน | 30–60 นาที |
| ศุกร์ | เช็กชื่อใหม่จากประกาศที่ลูกค้าลง → เพิ่มโครงการที่ยังไม่มี | 20 นาที |
| เดือนละครั้ง | รัน SQL นับจำนวน + ทดสอบค้นหา 10 ชื่อยอดนิยม | 15 นาที |

---

## เมื่อเจอโครงการที่ “ดึงไม่ได้”

1. กด **เพิ่มโครงการ** มือ — ใส่ชื่อ + เขต + พิกัดจาก Google Maps (คัดลอกพิกัดจากแผนที่)
2. ใส่ **ชื่อเล่น** ใน alias ให้ครบ
3. บันทึก → ทดสอบค้นหาในลงประกาศอีกครั้ง

---

## สรุปหนึ่งบรรทัด

**ทาง A = เติมสมุดชื่อโครงการในแอดมิน → แท็บ「โครงการ」→ ไล่ทีละโซนไม่ให้ตกหล่น → ทดสอบที่ลงประกาศ**

ไม่ต้องใส่ Google API key
