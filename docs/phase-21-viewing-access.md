# Phase 21 — การเปิดห้องนัดดู (ตอนลงประกาศ)

## ตำแหน่งในแอป

ขั้น **หัวข้อและรายละเอียด** ของฟอร์มสร้างประกาศ — หลัง Line ID

## ชุดคำถาม (ไม่บังคับครบ)

1. **คำอธิบาย** — ไม่ต้องกรอกทุกช่อง ทีมจะสอบถามเมื่อมีลูกค้านัดดูจริง
2. **สอบถามภายหลัง** (สวิตช์, เปิดเป็นค่าเริ่มต้น) — แนะนำสำหรับประกาศใหม่
3. **เปิดห้องได้แบบไหน** (เลือกได้หลายข้อ):
   - เจ้าของมาเปิดเอง → แจ้งล่วงหน้า **1 หรือ 2 วัน**
   - เบิกคีย์การ์ด/กุญแจกับ **นิติบุคคล**
   - **ฝากกุญแจ** (ตู้จดหมาย / จุดรับกุญแจ)
4. **หมายเหตุ** (ไม่บังคับ, สูงสุด 200 ตัวอักษร)
