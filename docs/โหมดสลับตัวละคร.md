# โหมดสลับตัวละคร (หลังบ้าน)

## เข้าระบบ — บัญชีเดียวทุกคน

| รายการ | ค่า |
|--------|-----|
| อีเมล | `demo-admin@livingbkk.local` |
| รหัสผ่าน | `demo12345` |

ทุกคนในทีมใช้บัญชีนี้ล็อกอินก่อน แล้วค่อยสลับเป็นตัวละครของตัวเองภายใน

---

## สลับตัวละคร

1. เข้าหลังบ้าน (`/admin`)
2. กดไอคอน **บัญชี/ป้าย** มุมขวาบน หรือแถบ **กำลังเล่นเป็น: …**
3. ใส่ **รหัสตัวละคร** และ **รหัสผ่าน** (ใช้ค่าเดียวกัน)
4. กด **เข้าใช้งานในบทบาทนี้** หรือแตะชื่อในรายการ

**กฎง่าย ๆ:** รหัสผ่าน = รหัสตัวละคร (เช่น `ceo-01` / `ceo-01`)

---

## รายชื่อตัวละคร (78 คน)

| ตำแหน่ง | จำนวน | รหัสตัวอย่าง |
|---------|-------|----------------|
| CEO | 1 | `ceo-01` |
| SUP | 2 | `sup-01`, `sup-02` |
| Lead (รองจาก SUP) | 5 | `lead-01` … `lead-05` |
| Admin | 10 | `admin-01` … `admin-10` |
| เอเจ้นพานัด | 10 | `guide-01` … `guide-10` |
| ลูกค้า | 20 | `seeker-01` … `seeker-20` |
| โคนายหน้า | 10 | `broker-01` … `broker-10` |
| เจ้าของทรัพย์ | 20 | `owner-01` … `owner-20` |

---

## แต่ละบทบาทเห็นอะไร

| บทบาท | หลังบ้าน |
|--------|----------|
| CEO / SUP / Lead / Admin | เมนูหลังบ้านตามสิทธิ์ tier |
| เอเจ้นพานัด (`guide-xx`) | **ปฏิทินนัดชม** เฉพาะเคสที่มอบให้ |
| ลูกค้า / นายหน้า / เจ้าของ | แจ้งให้เปิดหน้าบ้านทดสอบ (อยู่ในแชทจำลองเป็นตัวละคร) |

---

## ทดสอบมอบหมายเอเจ้นพานัด

1. สลับเป็น **Admin** (`admin-01`)
2. เปิด **ปฏิทินนัดชม**
3. เลือกเคสที่ขึ้น **ยังไม่ระบุคนพา** (มีหลายเคสจำลอง)
4. กด **ระบุคนพา** → เลือก `guide-01` … `guide-10`
5. สลับเป็น `guide-03` → ควรเห็นเฉพาะเคสที่มอบให้คนนั้น

---

## แชทจำลอง

เมื่อ `ADMIN_DEMO_CASES=true` ระบบสร้างแชทตัวอย่างอัตโนมัติ:

- **รอรับงาน** — ยังไม่มีแอดมินรับ
- **งานของฉัน** — ผูกกับ `admin-xx` / `lead-xx` ตามตัวละครที่สลับอยู่
- ลูกค้า/นายหน้าในแชท = `seeker-xx` / `broker-xx`
