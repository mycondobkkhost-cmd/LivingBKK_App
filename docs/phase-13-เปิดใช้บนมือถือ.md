## สำหรับคุณ (3 ทางเลือก)

### ทางเลือก 1 — ทดลองบนมือถือในบ้าน (ง่ายสุด)

1. รันแอปบน Mac: `./scripts/run-app.sh`  
2. มือถือต่อ Wi‑Fi เดียวกับ Mac  
3. เปิดลิงก์ `http://IP-ของ-Mac:8084` (ดู IP ใน System Settings → Wi‑Fi)  
4. **Safari/Chrome บนมือถือ** → เมนู → **เพิ่มไปที่หน้าจอโฮม**

### ทางเลือก 2 — สร้างเว็บแล้วอัปโหลด (ให้คนอื่นเปิดได้)

```bash
./scripts/build-web.sh
```

ได้โฟลเดอร์ `mobile/build/web` → ลากขึ้น [Netlify Drop](https://app.netlify.com/drop) ฟรี → ได้ลิงก์ `https://xxx.netlify.app`

### ทางเลือก 3 — App Store / Play Store (เฟสถัดไป)

ต้องมี Mac รุ่นใหม่ + Xcode หรือบัญชี Google Play — **ยังไม่จำเป็นตอนนี้**
